  ---
🌟 アプリ概要
Lumina Connectは、言葉にしづらい「学びのプロセス」を光の演出で可視化する、教育支援ツールです。先生の手元からの操作で、生徒の画面に「悩み」「ヒント」「解決」の光を届け、クラスの一体感と学びの意欲を高めます。

主な機能
* **3段階の演出：** 「思考（グレー）」「ヒント（太陽）」「解決（星空）」の演出で生徒の心に寄り添います。
* **瞬間ログ機能：** いつ、どんな支援が行われたかを自動で記録。振り返りに活用できます。

🛡️ プライバシーポリシー
本アプリは、生徒の皆さんが安心して利用できるよう、以下の運用を徹底しています。
* **データの非送信：** 入力された情報や利用データが外部サーバーに送信されることはありません。
* **個人情報の不保持：** 氏名、メールアドレス、端末識別子など、個人を特定できる情報を一切取得・保持しません。
* **広告・分析・追跡の排除：** 広告表示やユーザー行動の分析、および追跡（トラッキング）目的でデータを使用することはありません。
